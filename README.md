# Zeiterfassung Testversionen

Dieses öffentliche Repository stellt freigegebene Windows-Testversionen der
Anwendung **Zeiterfassung** bereit. Es enthält keinen Quellcode und keine
Nutzerdaten.

## Download

- [Downloadseite](https://thomash100.github.io/Testversionen/)
- [Alle Test-Releases](https://github.com/Thomash100/Testversionen/releases)
- [Aktuelle Testversion v1.6.9-test.2](https://github.com/Thomash100/Testversionen/releases/tag/v1.6.9-test.2)

Jedes Release enthält, soweit verfügbar:

- Windows-Setup
- Portable-ZIP
- SHA-256-Prüfsummen
- kurze Test- und Änderungshinweise

## Hinweis

Die bereitgestellten Pakete sind Vorabversionen für kontrollierte Tests. Vor
dem Einsatz sollten bestehende Daten gesichert werden. Fehler bitte ohne
private Inhalte, Zugangsdaten oder personenbezogene Nutzerdaten melden.
